Key Features of HTML

1. Structure

HTML provides the basic structure of web pages using elements like <head>, <body>, <header>, <footer>, and more.

<header>Header</header>
<main>Main content</main>
<footer>Footer</footer>

Other Semantic Elements

HTML5 introduces semantic elements like <article>, <section>, <nav>, and <aside> for better structure and accessibility.

2. Text Formatting

HTML allows formatting text with elements such as <h1> to <h6> for headings, <p> for paragraphs, <strong> for bold, and <em> for italics.
